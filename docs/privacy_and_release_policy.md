# Privacy and Release Policy

This repository contains only anonymized and sanitized artifacts from the pilot study.

## Included artifacts

The repository may include:

- anonymized participant codes;
- generated prompts;
- generated LLM responses;
- methodological documentation;
- sanitized instrument descriptions;
- aggregate results;
- final article PDF;
- final presentation slides.

## Excluded artifacts

The repository should not include:

- real participant names;
- informal participant labels;
- raw IPIP-NEO/OCEAN answers;
- complete psychological profiles;
- raw participant test answers;
- open-ended responses that could identify participants;
- API keys;
- `.env` files;
- execution logs with sensitive metadata;
- institution-specific confidential information.

## Reason for sanitization

Raw test files and raw profile files may contain identifiable information, writing patterns, correction notes, and sensitive details.

For this reason, the public repository provides sanitized descriptions of the instruments, scoring structure, prompt strategy, execution protocol, and aggregate results.

## Reproducibility approach

The repository supports reproducibility through:

- prompt construction logic;
- generated prompts;
- generated responses;
- scoring summaries;
- aggregate results;
- documentation of the experimental procedure;
- documentation of contamination control.

The repository does not aim to publish all raw participant data.

## Scope of public release

The public release is sufficient to understand:

- what was tested;
- how the LLM materials were generated;
- how the conditions differed;
- how K0 and K1 were scored;
- what the corrected aggregate results were;
- what limitations remain.

It is not intended to expose identifiable participant-level raw data.
